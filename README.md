# Mapa-de-carreira

Este repositório contém um mapa de carreira desenvolvido para organizar objetivos profissionais, habilidades necessárias e a evolução ao longo do tempo.
A ideia é manter um registro claro do caminho desejado na carreira e dos passos necessários para alcançar cada etapa.

O conteúdo pode incluir metas de curto, médio e longo prazo, competências técnicas e comportamentais, além de trilhas de aprendizado.
